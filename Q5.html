Description list of full stack web development tech stack

HTML
Markup language used for structuring content on the web.
CSS
Style sheet language used for describing the presentation of a document.
JavaScript
Programming language for adding interactivity to web pages.
Node.js
JavaScript runtime for server-side development.
React
A JavaScript library for building user interfaces.
Express
Web application framework for Node.js, used for building APIs.
MongoDB
NoSQL database for storing data in a flexible, JSON-like format.
Git
Version control system for tracking changes in code.
Bootstrap
Front-end framework for developing responsive websites quickly.
REST
Architectural style for designing networked applications using HTTP requests.
