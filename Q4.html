1. <html>: The root element that contains all HTML content.
  2. <head>: Contains meta-information about the document, like title and links to stylesheets.
  3. <title>: Sets the title of the web page, displayed in the browser's title bar.
  4. <body>: Contains the main content of the web page that is visible to users.
  5. <h1> to <h6>: Header tags that define headings, with <h1> being the highest level.
  6. <p>: Represents a paragraph of text.
  7. <a>: Defines a hyperlink that links to another page or resource.
  8. <img>: Embeds an image in the document, with attributes for source and alt text.
  9. <ul>: Creates an unordered list of items, typically displayed with bullet points.
  10. <ol>: Creates an ordered list of items, displayed with numbers.
